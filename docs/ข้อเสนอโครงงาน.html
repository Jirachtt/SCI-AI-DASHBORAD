ข้อเสนอโครงงาน

1. ชื่อโครงงาน

ภาษาไทย: การพัฒนาระบบ Dashboard อัจฉริยะสำหรับคณะวิทยาศาสตร์ มหาวิทยาลัยแม่โจ้ โดยใช้เทคโนโลยี AI

ภาษาอังกฤษ: Development of Intelligent Dashboard System for Faculty of Science, Maejo University using AI Technology

2. หลักการและเหตุผล

คณะวิทยาศาสตร์ มหาวิทยาลัยแม่โจ้ มีข้อมูลจำนวนมากเกี่ยวกับการบริหารจัดการ ทั้งข้อมูลนักศึกษา บุคลากร งบประมาณ งานวิจัย และยุทธศาสตร์ ซึ่งข้อมูลเหล่านี้กระจัดกระจายอยู่ในหลายระบบ ทำให้ผู้บริหารต้องใช้เวลามากในการรวบรวมและวิเคราะห์ข้อมูล

ด้วยเหตุนี้ ผู้พัฒนาจึงได้นำเทคโนโลยีปัญญาประดิษฐ์ (AI) มาช่วยในการสร้างระบบ Dashboard ที่สามารถแสดงผลข้อมูลแบบ Interactive พร้อมทั้ง AI Chatbot ที่ช่วยวิเคราะห์ข้อมูลเชิงลึกและพยากรณ์แนวโน้มได้โดยอัตโนมัติ เพื่อช่วยให้ผู้บริหารตัดสินใจได้อย่างรวดเร็วและแม่นยำ

3. วัตถุประสงค์

  1. เพื่อพัฒนาระบบ Dashboard สำหรับรวบรวมและแสดงผลข้อมูล 6 ด้านของคณะวิทยาศาสตร์
  2. เพื่อพัฒนา AI Chatbot ด้วย Google Gemini API สำหรับการวิเคราะห์และพยากรณ์ข้อมูล
  3. เพื่อพัฒนาระบบแสดงผลข้อมูลแบบ Interactive พร้อมกราฟ 7 ประเภท
  4. เพื่อพัฒนาระบบ Role-Based Access Control สำหรับผู้ใช้ 4 ระดับ

4. ขอบเขตของโครงงาน

  1. เป็น Web Application (Single Page Application) พัฒนาด้วย React 19 + Vite 7
  2. ข้อมูล 6 หมวด: HR, Student, Research, Finance, OKR, AI Chat
  3. Authentication: Firebase (Email/Password + Google Sign-In)
  4. AI: Google Gemini 2.5 Flash API (ถาม-ตอบ, สร้างกราฟ, พยากรณ์)
  5. Charts: Chart.js (7 ประเภทกราฟ + Zoom/Pan)
  6. ใช้ข้อมูลจำลอง (Mock Data) ไม่เชื่อมต่อฐานข้อมูลจริงของมหาวิทยาลัย

5. เครื่องมือในการพัฒนา

  • React 19.2.0, Vite 7.3.1, React Router DOM 7.13.0
  • Firebase 12.9.0 (Authentication)
  • Chart.js 4.5.1 + react-chartjs-2 5.3.1 + chartjs-plugin-zoom 2.2.0
  • Lucide React 0.563.0 (Icons)
  • Google Gemini API (gemini-2.5-flash)
  • Visual Studio Code, GitHub, Node.js

6. ผลที่คาดว่าจะได้รับ

  1. ระบบ Dashboard ที่รวบรวมข้อมูลครบ 6 ด้าน แสดงผลเป็นกราฟและตาราง
  2. AI Chatbot ที่สามารถตอบคำถาม สร้างกราฟ และพยากรณ์ข้อมูลได้
  3. ระบบควบคุมสิทธิ์ 4 ระดับที่ทำงานถูกต้อง
  4. ต้นแบบ (Prototype) ที่สามารถต่อยอดกับข้อมูลจริงได้ในอนาคต

7. แผนการดำเนินงาน (Timeline)
